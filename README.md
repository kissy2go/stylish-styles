# Stylish styles

## Requirements

* [Node.js](https://nodejs.org/)
* [Gulp](http://gulpjs.com/)
* [Sass](http://sass-lang.com/)

## Installation

Run following command after clone repository:

```sh
$ npm install
```

## Gulp tasks

### `clean`

Cleanup working directory.

### `compile`

Compile SCSS files.

### `watch`

Watch source files and auto compile.
